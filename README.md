## About This Portfolio

This portfolio documents my practical engagement with the Networks and System Security module throughout the academic year. It includes weekly laboratory exercises, code samples, security implementations, and demonstrations of my learning journey in information security.

## Portfolio Structure

Each lab contains:

- **Practical exercises:** Hands-on security implementations and testing
- **Code samples:** Scripts, tools, and security-related programs
- **Research:** Analysis of security concepts, vulnerabilities, and best practices
- **Documentation:** Jupyter notebooks with explanations and outputs

## Lab Contents

### [Lab 2: Public Key Cryptography & RSA](Lab%202/)
- RSA key generation using Python's cryptography library
- Public/private key pair creation and serialization
- Implementation of asymmetric encryption principles
- File-based key storage (PEM format)

### [Lab 3: Password Security & Strength Meter](Lab%203/)
- Password strength evaluation algorithm
- Entropy calculation for password complexity
- Character variety analysis (lowercase, uppercase, digits, symbols)
- Security feedback and scoring system

### [Lab 4: File Integrity & Hashing](Lab%204/)
- SHA-256 hash generation for files
- Automated hash computation across directory structures
- CSV-based hash result tracking with timestamps
- File integrity verification implementation

### [Lab 5: Web Application Security Testing](Lab%205/)
- Wapiti web vulnerability scanner integration
- Automated security assessment of web applications
- HTML report generation and analysis
- Vulnerability detection and reporting

### [Lab 6: Malware Analysis Fundamentals](Lab%206/)
- File hash computation (MD5, SHA1, SHA256)
- String extraction from binary files
- PE file analysis using pefile library
- YARA rule integration for malware detection

### [Lab 7: Network Reconnaissance & Port Scanning](Lab%207/)
- Domain information gathering (DNS, WHOIS)
- IP address resolution and geolocation
- Port scanning implementation
- Network security assessment techniques

### [Lab 9: AI & LLM Security Applications](Lab%209/)
- Local LLM deployment with Ollama
- Integration of language models for security tasks
- Performance benchmarking of AI models
- Practical AI applications in cybersecurity
